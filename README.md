# react-redux-boilerplate
Simple React Redux Boilerplate

<br>
## Available Scripts

#### Install dependencies:
##### `make run.install`

----
For running development environment, run in 2 different terminals:

#### Run server:
##### `make run.server`
#### Run react:
##### `make run.react`
----
#### Build project:
##### `make run.build`

Builds the app for production into the `build` folder

----

#### Clean:
##### `make run.clean`
----
